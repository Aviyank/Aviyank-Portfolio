# Profile Pictures Folder

## How to Add Your Profile Picture

1. **Save your anime portrait image** in this folder with the name `profile-picture.png`
   - Supported formats: PNG, JPG, JPEG, SVG, WebP
   - Recommended size: 300x300 pixels or larger
   - Recommended format: PNG with transparent background

2. **File naming conventions:**
   - Main profile picture: `profile-picture.png` (or .jpg, .svg, etc.)
   - Alternative names: `avatar.png`, `me.png`, `portrait.png`

3. **The project will automatically:**
   - Copy your image to the correct location
   - Update the website to display your new profile picture
   - Handle different image formats

## Supported Image Formats
- PNG (recommended for transparency)
- JPG/JPEG (good for photos)
- SVG (vector graphics)
- WebP (modern format with good compression)

## Image Requirements
- Minimum size: 200x200 pixels
- Recommended size: 300x300 pixels
- Maximum size: 1024x1024 pixels
- File size: Under 2MB for best performance

## What Happens Next
Once you add your image file:
1. The project will automatically detect it
2. Copy it to the correct static files location
3. Update your portfolio website to show your new profile picture
4. Apply proper styling (circular crop, borders, shadows)

## Troubleshooting
If your image doesn't appear:
1. Make sure the file is named correctly
2. Check that the file format is supported
3. Ensure the file size is reasonable
4. Restart the Django development server
